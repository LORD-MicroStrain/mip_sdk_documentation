Detailed Description

Controls which parameters are estimated by the Kalman Filter.

Desired settings should be logically ORed together.

Examples:

0x0001 - Enable Gyro Bias Estimation Only 0x0063 - Enable Gyro Bias, Accel Bias, and Mag Auto Hard and Soft Iron Cal States Only
