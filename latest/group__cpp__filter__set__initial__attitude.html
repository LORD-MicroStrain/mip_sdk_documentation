Detailed Description

Set the sensor initial attitude.

This command can only be issued in the "Init" state and should be used with a good estimate of the vehicle attitude. The Euler angles are the sensor body frame with respect to the NED frame.

The valid input ranges are as follows:

Roll: [-pi, pi] Pitch: [-pi/2, pi/2] Heading: [-pi, pi]
