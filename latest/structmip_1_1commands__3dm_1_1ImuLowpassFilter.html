◆ target_descriptor

uint8_t mip::commands_3dm::ImuLowpassFilter::target_descriptor = 0

Field descriptor of filtered quantity within the Sensor data set. Supported values are accel (0x04), gyro (0x05), mag (0x06), and pressure (0x17), provided the data is supported by the device. Except with the READ function selector, this can be 0 to apply to all of the above quantities.
