Detailed Description

Tare the device orientation.

This function uses the current device orientation relative to the NED frame as the current sensor to vehicle transformation. This command is provided as a convenient way to set the sensor to vehicle frame transformation. The filter must be initialized and have a valid attitude output. If the attitude is not valid, an error will be returned.
