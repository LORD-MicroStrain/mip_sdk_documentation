Detailed Description

This example shows a basic setup to configure the attitude filter with magnetometer as the heading source to stream filter data for the 3DM-CV7-AHRS, and 3DM-GV7-AHRS using C++. This is not an exhaustive example of all settings for those devices. If this example does not meet your specific setup needs, please consult the MIP SDK API documentation for the proper commands.

Sets up two event triggers for monitoring Euler angles:

  1. Roll angle threshold (Trigger ID 1)
    • Monitors X-axis rotation
    • Triggers when the angle exceeds +/-45 degrees
  2. Pitch angle threshold (Trigger ID 2)
    • Monitors Y-axis rotation
    • Triggers when the angle exceeds +/-45 degrees
Parameters
_deviceReference to the initialized MIP device interface
